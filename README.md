# Angular2MongoExpress
Angular2 with Mongo, Express, Webpack and Gulp

<h3>Install:</h3>
<pre>- npm install</pre>

<h3>Run:</h3>
<pre>- npm start </pre>

<h3>Website:</h3>
<pre>- localhost:8080 (if you want change port see script start into package.jon)</pre>

<h3>Express</h3>
<pre>- localhost:4200</pre>
